ADC Individual Project - User Management System

Server is running successfully!

Available REST Endpoints

  • POST /rest/createaccount - Create a new user account
  • POST /rest/login - Login and receive JWT cookie
  • POST /rest/showusers - List all users (ADMIN/BOFFICER only)
  • POST /rest/deleteaccount - Delete a user account (ADMIN only)
  • POST /rest/modifyaccount - Modify user attributes
  • POST /rest/showsessions - Show active sessions (ADMIN only)
  • POST /rest/showuserrole - Show user role (ADMIN/BOFFICER only)
  • POST /rest/changerole - Change user role (ADMIN only)
  • POST /rest/changepassword - Change your password
  • POST /rest/logout - Logout and invalidate session
Note: All endpoints require authentication except /createaccount and /login.
Authentication is handled via HttpOnly cookie named session::adc containing JWT.
Use Postman or curl to test these endpoints.

Project built with Jersey (JAX-RS) and Google App Engine
